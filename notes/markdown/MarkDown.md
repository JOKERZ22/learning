# markdown 笔记

###  标题 
方法一：#+空格  一个#一级，从一级到六级  
方法二：文本换行后+= 一阶，文本换行后+- 二阶，缺点：只能使用二级

###  字体
斜体 一个星号包裹  
    例如：*斜体*  
粗体 俩个星号包裹  
    例如：**粗体**  
粗斜体 三个星号包裹  
    例如：***粗斜体***

### 链接
方法一：[]()英文中括号+英文小括号；中括号中为展示内容，小括号中为链接地址 地址后加空格+英文双引号添加链接title  
    例如：[哔哩哔哩](http://www.bilibili.com "bilibili")  
方法二：<> 英文尖括号中填入链接地址  
    例如：<http:bilibili.com>

### 图片
与链接类似，在[]前加上英文感叹号

### 后置引用链接或图片 
先[]展示内容，紧后[]填入别名，后面任意位置声明别名的链接地址   
    例如：[哔哩哔哩][url]
[url]:http:www.bilibili.com

### 脚注
[^] 中填入唯一id  
    例如：脚注[^1]
[^1]:脚注1

### 引用
一个>表示一层引用  
    例如：  
>一楼
>>二楼
>>
>>>三楼

### 代码块
方法一：数字键1左边的顿号键（中文）包裹内容  
例如：`代码块1`  
方法二：代码区块，连续四个空格  
例如：
    代码块2  
方法三：一个tab键  
例如：
    代码块3

### 分割线
三个星号或-或_  
例如：
***
---
___

### 删除线
前后各俩个波浪线  
例如：
~~不想学习~~

### 无序列表
星号 - + 加空格  
例如：
* 1
- 2
+ 3

### 有序列表
数字 . 加空格  
例如：

1. 有序列表1

### 多级列表
在列表符前后加空格  
例如：
* 第一级
  * 第二级


### 换行
俩个空格

### 缩进
&emsp;一个中文字符
&ensp;半个中文字符
&nbsp;四分之一中文字符

### 表格

用｜分隔单元格，第二行被用来分隔表头和普通单元格，且都填写大于等于三个-，:用来控制对齐方式。

| Left-aligned | Center-aligned | Right-aligned |
| :---         |     :---:      |          ---: |
| git status   | git status     | git status    |
| git diff     | git diff       | git diff      |
